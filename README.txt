GunSmith - Paper 1.21.4 Full Project ZIP
--------------------------------------------------
This archive contains the GunSmith plugin project prepared for Paper 1.21.4.
Note: gradle-wrapper.jar is a placeholder. If missing, generate it locally with Gradle:
  1) Install Gradle (https://gradle.org/install) OR ensure gradle is available.
  2) Run in project root: gradle wrapper --gradle-version 8.10.2
  3) Then run: .\gradlew.bat build -Dorg.gradle.java.home="C:\Path\To\jdk-21"

After build, JAR will be at build\libs\GunSmith-0.1.0.jar
